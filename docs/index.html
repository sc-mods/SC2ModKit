SC2ModToolkit

A modular toolkit for working with StarCraft II Mods in Node.js and the browser.

Repository & Installation

  • GitHub Repository
  • npm install sc2modkit

Demo Projects

  • Mod Viewer (Browser Demo)
  • Mod Merger Script (Node.js)
  • CLI Tool Example

Usage Examples

Load and merge mods:

import { SC2ModReader } from 'sc2modkit';

const reader = new SC2ModReader({ base: './mods/' });
const mergedMod = await reader.merge(['Core.SC2Mod', 'Gameplay.SC2Mod']);

View mod contents in browser:

  • Open Mod Viewer and drop in `.SC2Mod` files.
